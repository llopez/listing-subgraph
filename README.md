# Listing Subgraph

This is the subgraph for the Listing smart contracts

## Tech Stack

* Thegraph
* AssemblyScript

## Getting Started

### First, install the dependencies

```bash
yarn install
```

### Then, generate the code and run the tests

```bash
yarn codegen
yarn test
```
